# Caluladora Trigonométrica
Projeto 1 da disciplina de Algoritmos I do curso de Ciências da Computação - Unesp Bauru

Trata-se de um programa, desenvolvido na linguagem C, que executa o cálculo de funções trigonométricas, como seno, cosseno e tangente. O projeto conta com um menu de setas, cujo intuito é ser bastante intuitivo ao usuário.
